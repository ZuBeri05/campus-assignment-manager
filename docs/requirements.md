# 校园课程作业管理系统 — 需求说明（MVP）

## 1. 项目目标
面向高校课程作业管理的全流程系统，覆盖：发布 → 提交 → 规则校验 → 相似度检测 → 评分与评语 → 通知。

## 2. 角色
- 学生
- 老师
- 助教
- 管理员

## 3. 功能范围（MVP）
### 3.1 认证与权限
- 邮箱验证码登录
- 角色权限控制（RBAC）

### 3.2 课程与作业
- 课程创建/管理（老师/助教）
- 作业发布（标题、描述、截止时间、规则）
- 作业提交（文件上传 + 文本说明）
- 规则校验（文件类型、大小、必备文件）

### 3.3 相似度检测
- 基础文本相似度（示例：TF-IDF / Jaccard / SimHash）
- 生成相似度报告（仅老师/助教可见）

### 3.4 评分与评语
- 评分（数值）
- 评语

### 3.5 通知
- 站内消息

### 3.6 系统设置（管理员）
- SMTP 邮箱配置（主机、端口、账号、授权码、SSL/TLS）
- 可启用/禁用发送

## 4. 非功能性需求
- 前后端分离
- 可本地部署
- 数据库可迁移（MySQL 8）

## 5. MVP 验收标准
- 学生可登录并提交作业
- 教师可发布作业、评分、查看相似度
- 管理员可配置 SMTP 并发出验证码
