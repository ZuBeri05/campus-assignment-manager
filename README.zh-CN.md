# 校园课程作业管理系统（Campus Assignment Manager）

面向高校课程作业管理的全流程系统（MVP），覆盖：发布 → 提交 → 规则校验 → 相似度检测 → 评分与评语 → 通知。

## 技术栈
- 后端：Java 17 / Spring Boot 3 / MySQL 8
- 前端：Vue 3 / Vite / Element Plus

## 功能范围（MVP）
- 邮箱验证码登录（SMTP 由管理员后台配置）
- 课程与作业管理
- 作业提交 + 规则校验
- 相似度检测（基础版）
- 评分与评语
- 站内通知
- 管理员系统设置
