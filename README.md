# Campus Assignment Manager

MVP for a campus assignment management system (Software Engineering project).

## Tech Stack
- Backend: Java 17, Spring Boot 3, MySQL 8
- Frontend: Vue 3, Vite, Element Plus

## Modules (MVP)
- Email OTP login
- Courses & assignments
- Submission + rule validation
- Similarity detection (basic)
- Grading & feedback
- Admin SMTP settings
